# Vault – Secure Web3 Storage (Aptos)

Vault is an experimental Web3 storage project exploring decentralized and verifiable data storage on the Aptos ecosystem.

## Goal

To learn and experiment with blockchain-based storage systems and build simple verifiable data records on testnet.

## Features

- Store simple data hashes
- Verify stored records
- Testnet based experiments
- Aptos ecosystem exploration

## Tech Stack

- Aptos Blockchain
- Node.js
- JSON storage simulation

## Project Structure

app.js – main script  
vault.json – stored records  
config.json – network configuration

## Run

node app.js
